Un blog di HTML

HTML5 è l'ultima versione di HTML, il linguaggio di markup attualmente più diffuso sul web, che spesso si interseca con JavaScript.
Questa guida ci permette di conoscere ed applicare da subito le novità introdotte in quest'ultima specifica, partendo dai nuovi tag semantici, fino alle funzionalità più avanzate, come le nuove API JavaScript.
